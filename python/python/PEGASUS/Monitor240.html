980 PRO PCle 4.0 NVMe M.2 固態硬碟 1TB

高水準的 SSD 固態硬碟效能

Key Specifications:

  • 三星 PCIe 4.0 NVMe SSD 980 PRO 通過釋放其強大效能，為您提供高水準的計算效能。
    • 憑藉 PCIe 4.0 介面的優勢，980 PRO 實現兩倍於 PCIe 3.0 的資料傳輸速率，同時向下相容 PCIe 3.0，確保了更廣泛的應用。

    Additional Specifications:

  • 強悍的速度
    • 980 PRO 搭載了三星專為 PCIe 4.0 SSD 定制的 Elpis 控制器，對速度進行了優化，其讀取速度高達 7000 MB/s，比 PCIe 3.0 SSD 快 2 倍，比 SATA SSD 快 12.7 倍。980 Pro 在 PCIe 4.0 上實現的最高速度，在其他環境中可能會略有下降。
    • 980 PRO 專為遊戲鐵粉和技術發燒友設計，可為遊戲、圖形、資料分析等領域中的重負荷應用提供高性能頻寬和輸送量。遊戲載入速度迅捷，讓您玩得過癮。
    • 高效固態硬碟通常需要高效的散熱控制。為了確保穩定的效能，980 PRO 採用鍍鎳塗層以提高控制器的散熱效率，同時使用散熱標籤實現對 NAND 晶片的有效散熱控制。
